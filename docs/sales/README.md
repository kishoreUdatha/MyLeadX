## How to Use

### For Customer-Facing Use
- **Product Brochure**: Share with new prospects
- **Feature Guide**: Send to evaluators doing deep-dives
- **Pricing**: Share during commercial discussions
- **One-Pager**: Use at events, trade shows, quick meetings

### For Internal Sales Use
- **Competitive Comparison**: Prepare for competitive deals
- **Sales FAQ**: Handle objections and questions

### For Commercial / Legal
- **Purchase Order Template**: Issue to the customer once a deal is closed
- **Service Agreement**: Attach to the PO as the master terms for the subscription
- **Mutual NDA**: Execute before sharing confidential details during evaluation

## Document Formats

These documents are in Markdown format. To convert:

### To PDF
```bash
# Using pandoc
pandoc 01-product-brochure.md -o product-brochure.pdf

# Using VS Code
# Install "Markdown PDF" extension, right-click > Export
```

### To HTML
```bash
pandoc 01-product-brochure.md -o product-brochure.html
```

### To DOCX
```bash
pandoc 01-product-brochure.md -o product-brochure.docx
```

## Customization

Before sharing with customers, update:

1. **Contact Information**
   - Website URL
   - Email addresses
   - Phone numbers

2. **Pricing** (if changed)
   - Plan prices
   - Voice minute pricing
   - Add-on costs

3. **Customer Testimonials**
   - Add real customer quotes
   - Include case study links

4. **Screenshots**
   - Add actual product screenshots
   - Include demo videos
